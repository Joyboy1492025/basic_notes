# Python Learning Notebooks

This workspace contains a set of Python learning notebooks that introduce basic Python concepts with hands-on examples and exercises.

## Project Structure

- `notes/00_python_intro.ipynb` — Python introduction, setup, and first steps.
- `notes/01_Variables_And_Data_Types.ipynb` — Variables, data types, and simple assignments.
- `notes/02_Number_and_operators.ipynb` — Numeric values, arithmetic, and operator usage.
- `notes/03_strings.ipynb` — String creation, formatting, and manipulation.

## How to Use

1. Open this folder in VS Code.
2. Open the `notes/` notebooks individually.
3. Run each notebook cell to explore the concepts and see live Python output.

## Recommended Workflow

1. Start with `00_python_intro.ipynb` to get familiar with the environment.
2. Continue through the notebooks in order to build foundational skills.
3. Re-run and modify examples to practice and learn by doing.

## Notes

- These notebooks are designed for beginners learning Python.
- Use the built-in Python kernel in VS Code or Jupyter to execute the notebooks.
- Feel free to add new notebooks for additional topics as you learn.
