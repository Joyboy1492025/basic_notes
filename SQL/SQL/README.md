# SQL Learning Project

A comprehensive SQL learning repository covering foundational to advanced concepts, with practical scripts and detailed documentation.

## 📋 Project Overview

This project is designed as a complete learning resource for SQL, covering all major topics from basic queries to advanced techniques. It includes hands-on scripts, reference materials, and database backups for practice.

## 🎯 Learning Path

### Beginner
1. Start with **SQL_Complete_Theory_Guide.pdf** - Understand SQL basics
2. Review database backups in **database/** - Get familiar with sample data
3. Study **notes/01_DQL.pdf** & **Script nt/01_DQL.sql** - Learn SELECT queries and data retrieval

### Intermediate
4. Study **notes/02_DDL.pdf** & **Script nt/02_DDL.sql** - Learn table creation and modification
5. Study **notes/03_DML.pdf** & **Script nt/03_DML.sql** - Master INSERT, UPDATE, DELETE
6. Study **notes/04_JOIN.pdf** & **Script nt/04_JOIN.sql** / **04_JOIN_2.sql** - Master table joins
7. Study **notes/05_SET.pdf** & **Script nt/05_SET.sql** - Learn set operations

### Advanced
8. Study **notes/06_Single_Row_Functions.pdf** & **Script nt/06_Single_Row_Functions.sql** - Master single row functions

### Advanced
9. Study **07_Row_Level_Functions.pdf** - Learn built-in functions
10. Study **08_Aggregation_Analytical_Functions.pdf** - Master GROUP BY, aggregate functions
11. Study **09_Advanced_SQL_Techniques.pdf** - Explore advanced topics
12. Study **10_Performance_Optimization.pdf** - Learn optimization techniques
13. Study **10_SQL_30_Performance_Tips.pdf** - Practical performance tips
14. Study **11_AI_and_SQL.pdf** - Explore AI integration with SQL
15. Study **12_SQL_Projects*.pdf** - Work on real-world projects

## 📝 SQL Script Categories

### DQL (Data Query Language) - `01_DQL.sql`
Used for retrieving data from the database:
- SELECT statements
- WHERE clauses
- JOINs
- Subqueries
- Aggregations

### DDL (Data Definition Language) - `02_DDL.sql`
Used for defining and modifying database structure:
- CREATE TABLE
- ALTER TABLE
- DROP TABLE
- CREATE INDEX
- Constraints (PRIMARY KEY, FOREIGN KEY, etc.)

### DML (Data Manipulation Language) - `03_DML.sql`
Used for managing data in the database:
- INSERT
- UPDATE
- DELETE
- MERGE
